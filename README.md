⚡ Smart Electricity Bill Calculator
📌 Project Overview

The Smart Electricity Bill Calculator is a web-based application that helps users estimate their electricity consumption and calculate their electricity bill based on the usage of different household appliances.

Users can enter the number of hours appliances such as fans, televisions, air conditioners, and lights are used. The system calculates the electricity units consumed and displays the estimated bill. It also provides a visual graph of electricity usage and stores previous bill calculations as history.

This project is designed to help users understand and manage their electricity consumption more effectively.

🚀 Features

🔌 Appliance-based electricity usage calculation

📊 Electricity usage graph visualization

💰 Automatic electricity bill estimation

💾 Bill history saved using browser storage

🖼 Background image based UI

💻 Simple and interactive web interface

📊 How It Works

The user enters the number of hours each appliance is used.

The system calculates electricity units using appliance power ratings.

Total electricity consumption is calculated in kWh (units).

The bill is estimated based on a predefined electricity rate.

A graph displays electricity usage by appliance.

The bill history is saved in the browser for future reference.
